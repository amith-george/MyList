# MyList
This is a microproject that I have worked on seperately alone. It's a site that is mainly focusing on personalized lists to keep track of movies, tv shows, animes etc.
It is currently run using a temporary database which means the data that you enter will get deleted and will not be permanent.
Considering moving it to a permanent database in the future.
